## Installation et Démarrage

### Prérequis
- Node.js 20+
- PostgreSQL 15+
- Redis 7+

### Installation
```bash
# Cloner le dépôt
git clone https://github.com/remiv1/sauvetage-xxx.git
cd sauvetage-xxx

# Installer les dépendances
npm install

# Configurer l'environnement
cp .env.example .env
# Éditer .env avec vos valeurs

# Migrations de base de données
npm run db:migrate

# Seeds (données de test)
npm run db:seed
```

### Développement
```bash
# Démarrer en mode développement
npm run dev

# Le service sera accessible sur http://localhost:3000
```

### Tests
```bash
# Tests unitaires
npm test

# Tests avec couverture
npm run test:coverage

# Tests e2e
npm run test:e2e
```

### Build et Production
```bash
# Build
npm run build

# Démarrer en production
npm start
```
